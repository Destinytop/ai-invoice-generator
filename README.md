# AI Invoice Generator

A professional AI-powered invoice generation tool for freelancers, small businesses, and entrepreneurs.

## Features

- ⚡ Create invoices in under 30 seconds
- 🎨 Professional PDF export
- 💳 Payment integration (Stripe, PayPal)
- 🏢 Custom branding
- 📊 Automatic tax calculations
- 🔒 Secure and private

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Stripe (for payments)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Deployment

```bash
vercel --prod
```
